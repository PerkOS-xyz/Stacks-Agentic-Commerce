# Stacks Agentic Commerce - App

Next.js frontend with App Router.

## Structure

```
App/
├── src/
│   ├── app/          # Pages (Home, Agents, Jobs)
│   ├── components/   # UI components
│   └── utils/        # Helper functions
├── package.json
├── tsconfig.json
└── tailwind.config.js
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```
